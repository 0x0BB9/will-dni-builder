# will dni builder

## Project setup

```bash
pnpm install

# npm install
# yarn install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://github.com/0x0BB9/will-browser-data
cd will-browser-data
npm i
npm run dev
```
